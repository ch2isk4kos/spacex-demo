#React with GraphQL: SpaceX Demo

A React Application that fetches data from the SpaceX API using GraphQL. 

## Server

**Server Packages**:

* graphql
* express-graphql
* express
* axios
* nodemon
* cors
* classnames
* moment
* react-moment

## Client

**Client Packages**:

* @apollo-client
* graphql
